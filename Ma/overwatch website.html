Overwatch

Overwatch is a team-based multiplayer first-person shooter video game developed and published by Blizzard Entertainment. It was released in May 2016 for Microsoft Windows, PlayStation 4, and Xbox One.

Point Of The Game

There are five different game modes: Quick Play, Arcade, Competitve, Custom Game and AI. Quick Play: This game mode is about capturing/defending an objective or escort/stop the payload. Arcade: This game mode has five other different games modes. There is 3v3, 1v1, Mystery Brawls, No Limits, All Brawls and 6v6 Mystery Heroes.
